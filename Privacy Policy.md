# Privacy Policy for GitHub Heatmap Colorizer

Last updated: 2024-09-07

## Introduction

GitHub Heatmap Colorizer is a Chrome extension designed to enhance the visual appearance of the contribution calendar on GitHub user profile pages. We are committed to protecting your privacy and ensuring you feel confident while using our extension.

## Information Collection and Use

GitHub Heatmap Colorizer does not collect, store, or transmit any personal information or data from its users or from the GitHub pages it modifies. The extension operates entirely within your browser and only interacts with the GitHub website's DOM (Document Object Model) to modify the appearance of the contribution calendar.

### What information we access:

- The extension accesses the HTML content of GitHub pages, specifically the elements related to the contribution calendar.
- It reads the contribution data from tooltips associated with each day in the calendar.

### How we use this information:

- The accessed information is used solely to apply custom colors to the calendar cells based on the contribution count.
- This information is processed in real-time within your browser and is not stored or transmitted in any way.

## Data Storage

GitHub Heatmap Colorizer does not store any data, either locally on your device or remotely on servers.

## Third-party Access

We do not share any information with third parties because we do not collect or store any information.

## Changes to the Contribution Calendar

The only change made by this extension is to the visual appearance of the GitHub contribution calendar. It does not alter any actual data on GitHub or affect your GitHub account in any way.
